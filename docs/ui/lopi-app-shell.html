App shell — off-canvas sidebar

Every destination that used to live in a horizontal top-tab bar now lives in this left sidebar. It is closed by default on every load (translateX(-100%)), opened by the hamburger, and closes on scrim-click, Esc, or selecting a tab. The app's index route (/) now redirects to /stacks — Loop Stacks is the default view. Forge (the old /) moved to /forge.

This is the fully-hidden variant — SIDEBAR_MODE = 'hidden'. The icon-rail variant below is kept as CSS only, gated behind the same constant, so switching later is a one-line flip, not a rebuild.
