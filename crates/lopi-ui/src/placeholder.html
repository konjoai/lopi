forge ui not built

The lopi server is running. The new SvelteKit-based Forge UI hasn't been built yet — only the API endpoints are available right now.

To enable the full visualization (the Forge sphere, the Constellation, the live agent stream):

cd web
npm install
npm run build
# then restart lopi sail

The build produces static assets in web/dist/ which lopi-ui embeds into the binary via rust-embed. After rebuilding the Rust binary, the full UI will be served from this URL.
